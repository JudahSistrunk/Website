Games

Multicat:

Multicat is a game in which a group of cats try to take over the world using the power of cat videos. Guide a cat through the internet while collecting Likes to gain more cats. This game is a puzzle platformer. It is not finished, but a lot of the source files were lost in an unfortunate computer crash. Thus, work on it has stopped for the time being.

Weapon Globber:

Globber is a game that was created as part of a game dev competition. This is a game in which the player must explore procedurally generated dungeons in search of odd items to use as weapons. The player must find the way to the end of each room to progress. Your score is dependent on how far the you make it. See how far you can go. The webplayer has been having issues. If you can't play it, you can find the source files on GitHub here.

Multicat Survive:

Multicat Survive is a smaller version of Multicat using Multicat assets. In this version, the player must collect like buttons to get more cats. When a like button is collected, a dislike button is spawned at a random location. If a cat hits a dislike button, the player loses. Try to see how many cats you can get.

Hex Tap:

Hex Tap is an android game in which the player must tap hexagons as they show up on the screen as fast as possible. The game ends if a single hexagon is on screen for too long. Score is based on how many hexagons the player tapped. This game was created in Unity. It uses Unity ads.

Checksum:

Checksum is an unfinished game. It is on the play store, but it doesn't scale propperly on all devices. The player is presented with a grid of "bits" with row and collumn headers that represent the even or odd sum of the column or row. The player must fill in the grid with ones and zeros to satisfy the row/column conditions. Score is based off of speed and ratio of ones to zeros. It was put on the Play store, but it still needs work in order to propperly scale on all devices.
